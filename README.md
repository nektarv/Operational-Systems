# Operational Systems

This repository contains laboratory exercises developed for the **Operational Systems** course at the School of **Electrical and Computer Engineering, NTUA**.

The exercises are implemented in C on Linux/POSIX systems and cover fundamental operating systems concepts including:

- Process creation and management
- Inter-process communication
- UNIX signals
- Process supervision
- TCP socket programming
- Client-server communication

---

# Repository Structure

| Lab | Topic |
|---|---|
| `lab1` | Process creation with `fork()` and file operations |
| `lab2` | Signal handling and process supervision |
| `lab3` | Advanced process management using signals |
| `lab4` | TCP client implementation with sockets |

---

# Requirements

- GCC
- Linux or POSIX-compatible environment
